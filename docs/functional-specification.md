# TUI Media Player 機能仕様書

## 1. 概要

英語学習専用の最小限TUI音楽プレイヤーの機能仕様。

## 2. 基本機能仕様

### 2.1 ファイル管理

#### ファイル検索
**入力**: ディレクトリパス（コマンドライン引数）
**処理**: MP3ファイルのみを検索、自然順序ソート
**出力**: ファイルリスト

#### ファイル選択
**入力**: ↑↓キー、Enter
**処理**: 選択インデックス管理、ハイライト表示
**出力**: 選択ファイル

### 2.2 音声再生

#### 基本再生
**入力**: 選択ファイル、Spaceキー
**処理**: MP3デコード・再生、一時停止/再開
**出力**: 再生状態

#### 自動リピート
**入力**: なし（自動）
**処理**: ファイル終了検出→同一ファイル再開
**出力**: 継続再生

### 2.3 表示機能

#### 進捗表示
**入力**: 再生位置情報
**処理**: 時間フォーマット、進捗率計算
**出力**: 「02:34 / 05:12」「■■■■░░░░」

#### 状態表示
**入力**: アプリケーション状態
**処理**: ファイル名、再生状態の整形
**出力**: UI更新

## 3. データ構造

```rust
struct FileInfo {
    path: PathBuf,
    name: String,
}

struct AppState {
    files: Vec<FileInfo>,
    selected_index: usize,
    is_playing: bool,
    current_time: f64,
    total_time: f64,
}
```

## 4. キーバインド

| キー | 動作 |
|------|------|
| ↑ | 前のファイル |
| ↓ | 次のファイル |
| Enter | 選択ファイル再生 |
| Space | 再生/停止 |
| q | 終了 |
