# TUI Media Player 要件定義書

## 1. プロジェクト概要

### プロダクト名
TUI Media Player

### 目的
英語学習用音声ファイル（MP3）をCLIで再生するシンプルなツール

### 対象ユーザー
akiko.kusaka（英語学習者、Mac環境、CLI使い慣れている）

## 2. 機能要件

### FR-001: 音楽ファイル検索・表示
- 指定ディレクトリ内のMP3ファイルを一覧表示
- 日本語ファイル名の正常表示
- 自然順序でのソート

### FR-002: ファイル選択・再生
- ↑↓キーでファイル選択
- Enterキーで再生開始
- 現在選択中ファイルのハイライト表示

### FR-003: 基本再生制御
- Spaceキーで再生/一時停止切り替え
- 再生状態の視覚的表示

### FR-004: 自動リピート再生
- ファイル終了時の自動リピート（デフォルトON）
- 他ファイル選択で新しいファイルに切り替え可能

### FR-005: 進捗表示
- 現在時間/総時間の表示
- 進行状況バーの表示

### FR-006: アプリケーション制御
- qキーで即座終了
- コマンドライン引数でディレクトリ指定

## 3. 非機能要件

### NFR-001: 起動速度
- 起動から音声再生開始まで30秒以内

### NFR-002: 対応環境
- macOS 10.14以降
- MP3ファイルのみ対応
- UTF-8文字エンコーディング

### NFR-003: UI要件
- ターミナル最小80x24文字
- 直感的なキーバインド

## 4. 制約・除外事項

### 除外機能
- 音量制御（OS側で調整）
- プレイリスト機能
- シーク機能（早送り・巻き戻し）
- 設定ファイル
- ログ機能
- 複雑なエラーハンドリング

## 5. 受け入れ基準

- [ ] 指定ディレクトリのMP3ファイルが一覧表示される
- [ ] キーボードでファイル選択・再生ができる
- [ ] 日本語ファイル名が正しく表示される
- [ ] ファイル終了時に自動リピートされる
- [ ] 進捗が視覚的に分かる
- [ ] qキーで即座に終了できる
- [ ] 30秒以内に学習開始できる
