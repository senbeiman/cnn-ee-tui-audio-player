## 2. 英語学習特化ユーザーストーリー

### 2.1 学習ファイル管理ストーリー

**US-001: 英語学習ファイルの発見**
```
As a 英語学習者
I want to 学習用音声ファイルを一覧で確認したい
So that 今日学習する教材をすぐに見つけて選択できる
```

**受け入れ基準**:
- [ ] MP3ファイルのみが表示される
- [ ] 日本語のファイル名（例：「01　基礎トレーニング1(英→日) 2.mp3」）が正しく表示される
- [ ] ファイルが学習しやすい順序で並んでいる
- [ ] Downloads または指定した学習フォルダからファイルを読み込める

**US-002: 任意ファイルの素早い選択**
```
As a 英語学習者
```

**受け入れ基準**:
- [ ] ↑↓キーでファイルリストを自由に移動できる

- [ ] Enterキーで即座に再生開始できる
- [ ] 連番以外のファイル（例：13番の次に05番）も簡単に選択できる

**US-003: 学習に集中できる再生制御**

**受け入れ基準**:
- [ ] スペースキーで瞬時に再生/停止を切り替えられる

- [ ] 操作が学習の集中を妨げない
- [ ] 短い音声ファイルでもスムーズに再生される

## 3. 英語学習シナリオ

### 3.1 日常的な英語学習シナリオ
**シナリオ**: 通勤前の朝の英語学習ルーチン

1. **準備**: MacBookを開き、ターミナルを起動
2. **起動**: `tui-player ~/Downloads/2603/音声DL/`でプレイヤー起動
3. **選択**: 今日の学習目標「ダイジェスト3」を↑↓キーで選択
4. **再生**: Enterで再生開始、英語音声に集中
5. **反復**: 理解できなかった部分はrキーでリピート再生
6. **進行**: 理解できたら↓キーで次のファイル「ダイジェスト4」へ
7. **終了**: 時間になったらqキーで学習終了

**期待する体験**:
- 学習開始まで1分以内
- ファイル名（日本語）が読みやすく表示される
- 操作に迷わずスムーズに学習に集中できる

### 3.2 集中的な復習シナリオ
**シナリオ**: 苦手な部分の集中的な復習

1. **起動**: いつものコマンドでプレイヤー起動
2. **選択**: 「基礎トレーニング1(英→日)」を選択（連番でない任意選択）
3. **反復学習**: rキーでリピート再生オン、理解できるまで繰り返し
4. **切り替え**: 理解できたら「基礎トレーニング1(日→英)」に切り替え
5. **比較学習**: 英→日と日→英を交互に聞いて理解度を確認
6. **次へ**: 十分理解できたら基礎トレーニング2へ移行
7. **記録**: 今日学習した内容を頭の中で整理して終了

**期待する体験**:
- 任意の順序でファイルを選択できる柔軟性
- リピート機能で理解できるまで聞ける
- 学習フローを妨げない軽快な操作

### 3.3 通し学習シナリオ
**シナリオ**: 学習の総復習セッション

1. **起動**: 学習フォルダを指定してプレイヤー起動
2. **全体確認**: ファイルリストで全体の構成を確認
3. **順次再生**: 「通し音声(英→日)」から開始
4. **進捗管理**: 進行状況バーで全体の進度を確認
5. **必要時停止**: 理解不十分な箇所で一時停止、別ファイルで詳細学習
6. **継続**: 「通し音声(日→英)」で逆方向の理解度確認
7. **完了**: 全体を通して学習目標達成を確認

**期待する体験**:
- 長い通し音声でも安定して再生される
- 進捗が視覚的に分かりやすい
- 必要に応じて詳細学習に分岐できる柔軟性

```
学習意欲 → 起動 → 理解 → 学習開始 → 習慣化
   ↓       ↓     ↓       ↓       ↓
 やる気   簡単   直感的   効率的   継続
```

**各段階での重要ポイント**:

**最適化ポイント**:
