# Steerings - 機能別仕様書

TUI Media Playerの新機能や変更に関する仕様書を機能単位で管理します。

## ディレクトリ構成

各機能は連番プレフィックス付きのディレクトリで管理されます：

```
steerings/
├── 001-directory-navigation/    # ディレクトリナビゲーション機能
│   ├── README.md               # 機能概要
│   ├── directory-navigation.md # 詳細仕様
│   └── implementation-plan.md  # 実装計画
├── 002-repeat-toggle/          # リピート切り替え機能
│   ├── README.md               # 機能概要
│   └── repeat-toggle.md        # 詳細仕様
├── 003-continuous-playback/    # 連続再生機能
│   ├── README.md               # 機能概要
│   └── continuous-playback.md  # 詳細仕様
├── 004-title-group-playback/    # タイトル単位再生
│   ├── README.md               # 機能概要
│   └── title-group-playback.md # 詳細仕様
└── README.md                   # このファイル
```

## 機能一覧

### 001 - ディレクトリナビゲーション
**状態**: 実装完了 ✅

ディレクトリ階層を移動してファイルを選択する機能。Escキーで上位移動、Spaceキーでディレクトリ選択移動を実装。

**主要変更**:
- `FileInfo`にファイルタイプ追加
- `App`にディレクトリ状態管理追加
- Escキーによる上位移動
- UI表示でディレクトリ識別

### 002 - リピート再生機能
**状態**: 実装完了 ✅

rキーで選択中のファイルをリピート再生で開始する機能。グローバル設定なしのシンプルな仕様。

**主要変更**:
- 再生メソッドを分離（通常/リピート）
- rキーで直接リピート再生開始
- ファイルリストアイコンで再生モード表示
- グローバル設定削除でUI簡素化

### 003 - 連続再生機能
**状態**: 実装完了 ✅

cキーで連続再生を開始し、トラック終了時に自動で次のファイルを再生する機能。

**主要変更**:
- PlaybackMode enum追加（Single/Repeat/Continuous）
- トラック終了の自動検知と次トラック遷移
- cキーで連続再生開始
- 連続再生専用アイコン表示（⏭️）

### 004 - タイトル単位再生
**状態**: 実装完了 ✅

pキーで選択中MP3から再生を開始し、開始時と同じタイトルのトラックだけを連続再生する機能。タイトルが変わる位置で自動停止する。

**主要変更**:
- 再生開始キーをpに統一
- s/r/cキーの再生機能を削除
- 先頭番号プレフィックスを除いたファイル名完全一致で同一タイトル判定
- Spaceキーは一時停止/再開専用のまま維持

## 仕様書作成ガイドライン

### ディレクトリ命名
- `{連番3桁}-{機能名}` 形式（例: `001-directory-navigation`）
- 機能名はハイフン区切りの小文字英数字

### 必須ファイル
- **README.md**: 機能概要とファイル構成
- **{機能名}.md**: 詳細仕様書
- **implementation-plan.md**: 実装計画（必要に応じて）

### 推奨ファイル
- **test-scenarios.md**: テストシナリオ
- **migration.md**: 既存機能への影響・移行手順
