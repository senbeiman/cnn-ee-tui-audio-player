# 002 - リピート再生機能

## 概要
rキーで選択中のファイルをリピート再生で開始する機能。グローバル設定なしのシンプルな仕様。

## 状態
**実装完了** ✅

## ファイル構成
- `README.md` - このファイル（機能概要）
- `repeat-toggle.md` - 詳細仕様書

## 主要変更点
- `Player`構造体に`current_playback_repeat`フィールド追加
- 再生メソッドを分離：`play()`（通常）と`play_repeat()`（リピート）
- rキーで直接リピート再生開始
- ファイルリストアイコンで再生モード表示

## 実装済み機能
- [x] rキーでリピート再生開始
- [x] Spaceキーで通常再生開始
- [x] 再生モード別アイコン表示（🔁/▶）
- [x] グローバル設定削除でシンプル化

## 最終仕様のキーバインド
- **Space**: 通常再生（1回のみ）または一時停止/再開
- **r**: リピート再生（無限ループ）で頭から開始

## 影響範囲
- `src/player.rs`: 再生メソッド分離、状態管理
- `src/app.rs`: リピート再生メソッド追加
- `src/main.rs`: rキーハンドリング変更
- `src/ui.rs`: アイコン表示ロジック更新
