# リピート再生機能

## 概要
rキーで選択中のファイルをリピート再生で開始する機能。グローバル設定は持たず、再生開始時にリピート/通常を選択する仕様。

## 要件
- rキー：選択中ファイルをリピート再生で頭から開始
- Spaceキー：選択中ファイルを通常再生（1回のみ）で開始
- 再生中にrキーを押すと現在のファイルがリピート再生で頭から再開
- 再生中のファイルのアイコンで再生モードを表示
  - 🔁：リピート再生中
  - ▶：通常再生中

## 実装方針

### 1. データ構造の変更
- `Player`構造体に`current_playback_repeat`フィールドを追加
- グローバルなリピート設定は不要

### 2. 再生ロジック
- `play()`: 通常再生（1回のみ）
- `play_repeat()`: リピート再生（`repeat_infinite()`使用）
- 再生開始時にリピート状態を記録

### 3. キーバインディング
- `r`キー：`play_selected_repeat()`メソッドを呼び出し
- `Space`キー：従来通り通常再生または一時停止/再開

### 4. UI表示
- ファイルリストで再生中アイテムのアイコン表示
- プレーヤー部分にはリピート状態表示なし（シンプル化）

## 影響範囲
- `src/player.rs`: 再生メソッドを分離、リピート状態管理
- `src/app.rs`: リピート再生メソッド追加
- `src/main.rs`: rキーのハンドリング変更
- `src/ui.rs`: アイコン表示ロジック、フッター説明更新
