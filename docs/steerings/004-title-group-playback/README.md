# 004 - タイトル単位再生

## 概要
再生操作を `p` キーに集約し、開始したトラックと同じタイトルのMP3だけを連続再生する。タイトルが変わる位置、または次のMP3が存在しない位置で自動停止する。

## 状態
**実装完了** ✅

## ファイル構成
- `README.md` - このファイル（機能概要）
- `title-group-playback.md` - 詳細仕様書

## 主要機能
- **pキー**: 選択中MP3からタイトル単位再生を開始
- **タイトル継続**: 次のMP3が開始トラックと同じタイトルなら自動再生
- **自動停止**: 次のMP3のタイトルが変わったら停止
- **Spaceキー**: 再生中トラックの一時停止/再開

## 最終的なキーバインディング
| キー | 機能 | 動作 |
|------|------|------|
| p | 再生 | 選択中MP3から同一タイトルの範囲だけ連続再生 |
| Space | 一時停止/再開 | 現在再生中トラックのみ制御 |
| Enter | ディレクトリ移動 | ディレクトリ選択時のみ |

## 削除する機能
- `s`キーの1回再生
- `r`キーのリピート再生
- `c`キーの全MP3連続再生
