# 連続再生機能

## 概要
1トラックの再生が終了したら自動で次のトラックを再生する連続再生モードの追加。ファイルリスト順に順次再生し、最後のトラックまで到達したら停止する。

## 要件

### 基本機能
- cキー：選択中ファイルから連続再生開始
- 現在トラック終了時に次のファイル（MP3のみ）を自動再生
- ディレクトリは自動スキップして次のMP3ファイルへ
- 最後のMP3ファイル終了時に連続再生終了
- 連続再生中に他のキー操作で通常の再生制御可能

### キーバインディング（最新仕様）
- **s**: 通常再生開始（1回のみ）
- **c**: 選択中ファイルから連続再生開始
- **r**: リピート再生開始（無限ループ）
- **Space**: 一時停止/再開のみ（現在再生中トラックのみ対象、カーソル位置無関係）
- **Enter**: ディレクトリ選択・移動
- **j/k**: ファイル選択（再生は継続）

### UI表示
- **連続再生中アイコン**: ⏭️（ファイルリストとヘッダー）
- **通常再生中アイコン**: ▶️
- **リピート再生中アイコン**: 🔁
- **停止中アイコン**: ⏸️

## 動作シナリオ（最新仕様）

### 正常系
1. ファイル3を選択してcキー押下
2. ファイル3が連続モードで再生開始（⏭️アイコン）
3. ファイル3終了時、自動でファイル4へ遷移（カーソルも自動移動）
4. 最後のMP3ファイル終了時、連続再生終了

### キー操作の相互作用
- **連続再生中にs押下**: 現在選択中ファイルを通常再生で開始（連続再生中断）
- **連続再生中にr押下**: 現在選択中ファイルをリピート再生で開始（連続再生中断）
- **連続再生中にc押下**: 現在選択中ファイルから新しい連続再生開始
- **連続再生中にSpace押下**: 現在再生中トラックの一時停止/再開（連続モード維持）
- **連続再生中にEnter押下**: ディレクトリなら移動（連続再生終了）

### エラー系・境界条件
- 最後のファイルで連続再生開始 → そのファイルのみ再生して終了
- 連続再生中にディレクトリ移動 → 連続再生終了
- MP3ファイルが1つだけ → 通常再生と同じ動作

### キーバインディング完全仕様
| キー | 動作 | 連続再生への影響 |
|------|------|------------------|
| s | 通常再生開始 | 中断（Single mode） |
| c | 連続再生開始 | 新しい連続再生開始 |
| r | リピート再生開始 | 中断（Repeat mode） |
| Space | 一時停止/再開のみ | 維持（一時停止/再開） |
| Enter | ディレクトリ移動のみ | 移動時は中断 |
