# 003 - 連続再生機能

## 概要
1トラック終了時に自動で次のトラックを再生する連続再生モードの追加。ファイルリスト順に順次再生し、最後まで到達したら停止する機能。

## 状態
**実装完了** ✅

## ファイル構成
- `README.md` - このファイル（機能概要）
- `continuous-playback.md` - 詳細仕様書

## 主要機能
- **cキー**: 選択中ファイルから連続再生開始
- **自動遷移**: トラック終了時に次のMP3ファイルへ自動移行
- **スキップ機能**: ディレクトリは自動スキップ
- **終了条件**: 最後のMP3ファイル終了時に停止

## 最終的なキーバインディング
| キー | 機能 | アイコン | 動作 |
|------|------|----------|------|
| s | 通常再生 | ▶️ | 1トラックのみ再生 |
| r | リピート再生 | 🔁 | 現在トラックを無限リピート |
| c | 連続再生 | ⏭️ | 次トラックへ自動遷移 |
| Space | 一時停止/再開 | - | 現在再生中トラックのみ制御 |
| Enter | ディレクトリ移動 | - | ディレクトリ選択時のみ |

## 重要な設計変更
- **Spaceキーの役割分離**: 一時停止/再開専用に変更（カーソル位置無関係）
- **sキーで通常再生**: 明示的な再生開始操作を追加
- **Enterキーでディレクトリ移動**: ディレクトリ操作を分離

## PlaybackMode統合
