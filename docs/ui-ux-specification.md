# TUI Media Player UI/UX仕様書

## 1. 設計原則

- **最小性**: 必要最小限の情報のみ表示
- **直感性**: 説明不要の操作
- **学習重視**: 英語学習の邪魔をしない

## 2. 画面レイアウト

### 基本レイアウト（80x24最小対応）
```
┌─ TUI Media Player ──────────────────────────┐ ← ヘッダー (1行)
│ ♪ 01 基礎トレーニング1(英→日) 2.mp3         │
├─────────────────────────────────────────────┤
│ [■■■■■░░░░░] 02:34 / 05:12  🔁          │ ← プレイヤー (1行)
├─────────────────────────────────────────────┤
│ > 01 基礎トレーニング1(英→日) 2.mp3         │ ← ファイルリスト
│   02 基礎トレーニング1(日→英).mp3           │   (残り行-2)
│   03 基礎トレーニング2(英→日).mp3           │
│   04 基礎トレーニング2(日→英).mp3           │
│   05 ダイジェスト1(英→日).mp3              │
│   ...                                       │
├─────────────────────────────────────────────┤
│ [q]終了  [↑↓]選択  [Enter]再生  [Space]停止 │ ← フッター (1行)
└─────────────────────────────────────────────┘
```

## 3. 各エリア仕様

### ヘッダー
**表示**:
- アプリ名 + 現在再生ファイル名
- 再生状態アイコン: `♪` (再生中) / `⏸` (一時停止)

### プレイヤー
**表示**:
- 進捗バー: `[■■■■■░░░░░]` (10文字固定)
- 時間: `02:34 / 05:12` (MM:SS形式)
- リピート: `🔁` (常時表示)

### ファイルリスト
**表示**:
- 選択中: `> ファイル名`
- 非選択: `  ファイル名`
- 日本語ファイル名対応

### フッター
**表示**: 基本的なキーバインドヘルプ

## 4. キーバインド

| キー | 動作 |
|------|------|
| `↑` | 前のファイル |
| `↓` | 次のファイル |
| `Enter` | 選択ファイル再生 |
| `Space` | 再生/一時停止 |
| `q` | 終了 |

## 5. 色彩仕様

- **背景**: デフォルト（黒）
- **選択中ファイル背景**: 青
- **選択中ファイルテキスト**: 白
- **非選択ファイル**: グレー
- **進捗バー**: シアン
- **再生アイコン**: 緑

## 6. 応答性

- **キー入力応答**: 100ms以内
- **画面更新**: 500ms間隔
- **差分描画**: 変更部分のみ更新

## 7. 日本語対応

- **文字エンコーディング**: UTF-8
- **ファイル名表示**: 全角文字2文字幅計算
- **表示幅**: 画面幅に応じて省略（`...`）
